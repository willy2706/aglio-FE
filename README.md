# aglio-FE

1. Install ionic
--------------- 
      npm install -g cordova ionic

2. Clone repo ini
--------------
      git clone https://github.com/willy2706/aglio-FE

3. Masuk ke repo
--------------- 
      cd aglio-FE

4. Hubungkan Android anda (minimal API 16/Android 4.1) via USB
--------------- 

5. Build
--------------- 
      ionic build android

6. Deploy ke Android
--------------- 
      ionic run android
